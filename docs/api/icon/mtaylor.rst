
mtaylor
=========================

        This function represents the `Taylor Grid <https://confluence.ecmwf.int/display/METV/Taylor+Grid>`_ icon in Metview's user interface.


.. py:function:: mtaylor(**kwargs)
  
    Description comes here!


    :param taylor_label: Label of the grid. The default is: correlation.
    :type taylor_label: str


    :param taylor_label_colour: Colour of the label. The possible values:

        * background
        The default is: navy.
    :type taylor_label_colour: str


    :param taylor_label_height: Hieght of the label. The default is: 0.35.
    :type taylor_label_height: number


    :param taylor_primary_grid_increment: Reference used  of the Standard deviation plotting. The default is: 0.5.
    :type taylor_primary_grid_increment: number


    :param taylor_primary_grid_line_colour: Colour used to plot the primary grid. The possible values:

        * background
        The default is: navy.
    :type taylor_primary_grid_line_colour: str


    :param taylor_primary_grid_line_thickness: Thickness used to plot the primary grid. The default is: 1.
    :type taylor_primary_grid_line_thickness: int


    :param taylor_primary_grid_line_style: Line Style used to plot the primary grid. The possible values:

        * solid
        * dash
        * dot
        * chain_dash
        * chain_dot
        The default is: solid.
    :type taylor_primary_grid_line_style: str


    :param taylor_primary_grid_reference: Reference used  of the Standard deviation plotting. The default is: 0.5.
    :type taylor_primary_grid_reference: number

        * background
        The default is: navy.
    :type taylor_reference_line_colour: str


    :param taylor_reference_line_thickness: Thickness used to plot the primary grid. The default is: 2.
    :type taylor_reference_line_thickness: int


    :param taylor_reference_line_style: Line Style used to plot the primary grid. The possible values:

        * solid
        * dash
        * dot
        * chain_dash
        * chain_dot
        The default is: solid.
    :type taylor_reference_line_style: str


    :param taylor_primary_label: Turn the labels (on/off) of the primary grid. The possible values:

        * on
        * off
        The default is: on.
    :type taylor_primary_label: str


    :param taylor_primary_label_colour: Colour of the  labels  of the primary grid. The possible values:

        * background
        The default is: navy.
    :type taylor_primary_label_colour: str


    :param taylor_primary_label_height: Height of the  labels  of the primary grid. The default is: 0.35.
    :type taylor_primary_label_height: number


    :param taylor_secondary_grid: turn on/off the secondaries lines for the grid. The possible values:

        * on
        * off
        The default is: off.
    :type taylor_secondary_grid: str


    :param taylor_secondary_grid_reference: Reference used  of the Standard deviation plotting. The default is: 0.5.
    :type taylor_secondary_grid_reference: number


    :param taylor_secondary_grid_increment: Reference used  of the Standard deviation plotting. The default is: 0.5.
    :type taylor_secondary_grid_increment: number


    :param taylor_secondary_grid_line_colour: Colour used to plot the primary grid. The possible values:

        * background
        The default is: navy.
    :type taylor_secondary_grid_line_colour: str


    :param taylor_secondary_grid_line_thickness: Thickness used to plot the primary grid. The default is: 1.
    :type taylor_secondary_grid_line_thickness: int


    :param taylor_secondary_grid_line_style: Line Style used to plot the primary grid. The possible values:

        * solid
        * dash
        * dot
        * chain_dash
        * chain_dot
        The default is: solid.
    :type taylor_secondary_grid_line_style: str


    :param taylor_secondary_label: Turn the labels (on/off) of the secondary grid. The possible values:

        * on
        * off
        The default is: on.
    :type taylor_secondary_label: str


    :param taylor_secondary_label_colour: Colour of the  labels  of the secondary grid. The possible values:

        * background
        The default is: navy.
    :type taylor_secondary_label_colour: str


    :param taylor_secondary_label_height: Height of the  labels  of the secondary grid. The default is: 0.35.
    :type taylor_secondary_label_height: number


    :rtype: None
