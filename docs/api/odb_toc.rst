
ODB functions
===============
This is the list of all the functions related to :class:`Odb` data.

.. list-table::
    :widths: 20 80
    :header-rows: 0


    * - :func:`odb_filter`
      - Filters ODB data

    * - :func:`odb_visualiser`
      - Defines visualisation for ODB data

    * - :func:`retrieve`
      - Retrieves data from MARS

    * - :func:`write`
      - Writes/appends the given data to file
